# 欢迎

💡这是一个非官方关于广州大学通用接口的仓库，制作了一个有效的通用接口，意想于提供一个“开放平台”API，供学校各个校园业务开发者提供便利。整合正方教务的一系列复杂接口，

> 本项目旨在让校园业务开发者能够有一个便携规范的调用接口
